2019-08-07
更换镜像
约 364 字 预计阅读 1 分钟

文章目录

  • 2.1 Windows
  • 2.2 Linux

1. 前言

Maven 默认的中央仓库速度慢，可以考虑换成阿里的镜像。修改方式主要有两种，全局修改和针对当前项目的修改。

2. 全局修改

2.1 Windows

Windows 下 maven 的 .m2 文件夹地址在 C:\User\****\.m2 (***为具体的用户名)

在该路径下新建一个 setting.xml 文件(如果不存在的话)

2.2 Linux

接下来操作与 Windows 一样

3. 针对当前项目修改

在项目的 pom.xml 文件内的 <repositories> 节点下面增加新节点
